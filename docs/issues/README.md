# Claude Code Issues

This directory contains working notes and documentation for contributing to [anthropics/claude-code](https://github.com/anthropics/claude-code) issues.

## What's Here

We track Claude Code issues in this repo, including:

- **Issues we created** - Bug reports and feature requests with reproduction scripts and analysis
- **Issues we're contributing to** - Existing issues we've pulled in to examine, analyze, or help resolve

## Issue Index

See the [issues/](../../issues/) directory for tracked issues.
